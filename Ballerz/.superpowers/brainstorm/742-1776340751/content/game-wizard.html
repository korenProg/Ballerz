Create Game — Wizard Flow

How many steps should creating a game take?

A

3 steps — Setup → Players → Teams

Step 1 – Setup: Date/time picker, optional location, team name + color for each side.
Step 2 – Pick Players: Scrollable roster, tap to select who's playing. Shows selected count.
Step 3 – Teams: Choose method (Random / Manual / Balanced), see result, swap players if needed. Confirm creates the game.
